# dog

# Summary

- discovered .git subdir in web app hosting a Backdrop CMS application
- git-dumper to get source code; mysql credentials retrieved from settings.php
- BackDropScan tool used to enumerate for usernames; ffuf faster with more threads; user tiffany uses the password found in settings.php
- utilised authenticated RCE exploit to get webshell on Backdrop CMS; reverse shell obtained as www-data
- password reused for user johncusack
- johncusack has sudo privilege to use bee; GTFObin to leverage bee to get root shell
