# sea

# Summary

- WonderCMS vulnerable to XSS; CVE-2023041425 RCE
- Fixed path error and path to reverse shell and got rce as user www-data
- database.js contains a user’s bcrypt hash; hashcat to crack; user amay compromised
- port 8080 serving a website in development
- the log_file http parameter from POST request is vulnerable to LFI and command injections
- web server hosted as root; reverse shell with nohup as separate process to get stable reverse shell as root
