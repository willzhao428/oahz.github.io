# Summary

- Visit web page to find searchor 2.4.0 as part of stack deck; vulnerable to RCE
- inject payload in query parameter; reverse shell as user svc
- find svc password in cwd .git/config file
- sudo privileges permits us to execute system-checkup.py; permits us to use docker-inspect
- docker-inspect containers environment variable to unveil password to gitea user
- find vhost in apache config file of gitea instance; log in as administrator with gitea password
- read source code for system-checkup.py; code uses relative path to execute full-checkup.sh
- create exploit named [full-checkup.](http://full-checkup.py)sh to set SUID on /bin//bash
- execute system-checkup with full-checkup argument; execute bash with -p; root shell
