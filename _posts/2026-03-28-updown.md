# Summary

- feroxbuster to fuzz for subdir; found /dev/.git
- git-dumper to restore source code from git
- .htacecss reveal special header value to access dev subdomain
- analysing source code, index.php allow upload of files (original purpose is to request URL to see if they are up/down) with file extension filters, gets uploaded to /uploads dir in a directory that’s named the md5 hash of the time() function
- checker.php also deletes files after all requests are completed ~30s; index.php has LFI vuln, vulnerable to php filters and appends .php to included files
- upload zipped up web shell, use LFI vuln to request for our webshell; 500 Internal server error; probably disallow php dangerous functions
- use custom script based on dfunc_bypasser as php script; upload; find out proc_open is allowed
- proc_open reverse shell; reverse shell as www-data
- access to user developer’s home dir in and dev share; SUID binary includes python2’s input() vulnerability; **import**(’os’).system(’bash’) to get shell as developer
- sudo privilege reveal easy_install can be executed as sudo; GTFObin; make bash SUID; root
